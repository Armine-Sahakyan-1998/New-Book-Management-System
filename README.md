# 📚 Book Management System

A simple Python console application for managing a personal book library. This project demonstrates the use of dictionaries, 
functions, loops, conditional statements, and user input in Python.

## Features

* ➕ Add new books to your library.
* 📖 View all books and their reading status.
* 🗑️ Delete books from the library.
* ✅ Update a book's reading status (Read / Not Read).
* 🚪 Exit the application through a simple menu.

## Technologies Used

* Python 3
* Dictionary data structure
* Functions
* While loops
* Conditional statements (`if`, `elif`, `else`)
* User input (`input()`)

## How It Works

When the program starts, a menu is displayed where the user can choose one of the available options:

1. Add a new book
2. View all books
3. Delete a book
4. Update a book's reading status
5. Exit the program

The application stores book titles in a Python dictionary, where each book is associated with its current reading status.

## Purpose

This project was created as a beginner-friendly Python practice project to improve programming skills, 
including working with dictionaries, functions, and building menu-driven console applications.
